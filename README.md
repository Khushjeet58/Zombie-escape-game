# 🧟 Zombie Escape Game

A terminal-based adventure game written in **C++** where every decision affects your fate.

This project was built as part of my C++ learning journey. Players must survive a zombie outbreak by making choices that lead to different storylines and endings.

---

## 🎮 Features

- Multiple branching story paths
- Interactive terminal gameplay
- Choice-based decisions
- Multiple endings
- Beginner-friendly C++ implementation

---

## 🕹️ Gameplay

You wake up trapped inside a building after a zombie outbreak.

Will you:
- ⚔️ Fight your way out?
- 🏃 Escape through the window?
- 🤫 Hide and wait?

Every decision changes your story.

---

## 🚀 How to Run

### Compile

```bash
g++ terminal_game.cpp -o zombie_game
```

### Run

```bash
./zombie_game
```
